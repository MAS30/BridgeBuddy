# BridgeBuddy

BridgeBuddy is a Django-based web application developed for the SEN204 Internet and Web Programming final project.

## Project Description

BridgeBuddy is a community-focused platform where users can explore helpful relationship phrases, guides, community discussions, and create posts in a clean and user-friendly environment.

The project demonstrates fullstack web development concepts using Django, templates, routing, database models, and dynamic content rendering.

---

## Features

- User-friendly homepage
- Community section
- Phrase collection page
- Guides section
- About page
- Dashboard page
- Create post functionality
- Django template inheritance
- Responsive UI
- Dynamic backend integration

---

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

---

## Project Structure

- core/ → Django project configuration
- main/ → Main application
- templates/ → HTML templates
- models.py → Database models
- views.py → Application logic
- urls.py → Routing system

## How to Run the Project

### 1. Clone Repository

```bash
git clone <your-github-link>
```

### 2. Navigate to Project

```bash
cd BridgeBuddy
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Start Development Server

```bash
python manage.py runserver
```

### 6. Open Browser

```text
http://127.0.0.1:8000/
```
